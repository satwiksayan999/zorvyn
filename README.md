# Zorvyn

A full-stack MVP for a video streaming platform with the following architecture:

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Video delivery: Cloudinary

## Setup

### Backend
1. Change into `backend`
2. Copy `.env.example` to `.env`
3. Install dependencies: `npm install`
4. Start server: `npm run dev`

### Frontend
1. Change into `frontend`
2. Install dependencies: `npm install`
3. Start app: `npm run dev`

## Features
- User auth with JWT
- Video upload to Cloudinary
- Video playback, comments, likes, subscriptions
- Search by title, tags, creator name
- Creator channel pages
- Infinite scroll video feed
- Responsive React UI
